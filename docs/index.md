# photoToDices

convert photo to Dices

### Using

1-create a virtualenv

```
python3 -m venv .venv
```

2-activate the virtualenv

```
. ./.venv/bin/activate
```

3-install requirment

```
pip install -r requrment.txt

```

4- run it :)

```
python app.py
```

![sample](img.jpg)
contain more than 3M dices
